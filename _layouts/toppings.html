What's this about?

So you've had a look at HTML and JavaScript, and you're getting pretty good at making websites (that look a little better than this)! But now, it's time to add a little more... flavour to them (and make something using these elements.) We've got the perfect opportunity for you to do so.

What will I learn?

  • How to use grids to make fun layouts
  • How to use flexboxes
  • How to use variables in CSS
  • How to make layouts that are responsive to screen size
  • How to animate in CSS
  • How to use margins and padding
  • How to create a gradient background

Requirements

  • Gotta be unique! Have fun with it :)
  • Gotta have one thing animated
  • Gotta have one thing that changes when the screen size is reduced, using @media (not just relative units!)
  • Gotta use containers
  • Gotta use a grid
  • Gotta either have a gradient background OR a flexbox (or both!)

So what do I get?

If your website fulfils all the requirements, you get $5 to buy toppings for your boba or ice cream!
