FAQ

What is Tailwind CSS?

Tailwind CSS is a utility-first CSS framework that provides low-level utility classes to build custom designs. It allows you to build your website without writing any CSS.

Can I use CSS in my project?

Tailwind CSS allows you to develop your website using only HTML and Tailwind CSS in most cases. Pure CSS may only be used where something can't be done without it, such as custom animations.
